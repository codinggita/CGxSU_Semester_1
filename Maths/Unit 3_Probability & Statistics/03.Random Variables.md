# Random Variables

## Learning Objectives

- Define random variables rigorously
- Distinguish between discrete and continuous random variables
- Master probability mass functions (PMF) and probability density functions (PDF)
- Understand cumulative distribution functions (CDF)
- Calculate probabilities using PMFs, PDFs, and CDFs
- Work with multiple examples of each type

## Prerequisites

- Sample spaces and events
- Probability axioms
- Basic calculus (for continuous random variables)

## Definitions and Theoretical Foundations

### Definition 4.1 (Random Variable)

A **random variable** is a function $X: S \to \mathbb{R}$ that assigns a real number to each outcome in the sample space $S$.

**Notation:** We write $X(\omega)$ or simply $X$ to denote the value assigned to outcome $\omega \in S$.

**Key Property:** For any set $B \subseteq \mathbb{R}$, the set $\{X \in B\} = \{\omega \in S : X(\omega) \in B\}$ must be an event.

**Interpretation:** Random variables numerically encode the outcomes of random experiments, allowing us to apply mathematical analysis.

### Definition 4.2 (Discrete Random Variable)

A random variable $X$ is **discrete** if it can take at most countably many values $\{x_1, x_2, x_3, \ldots\}$.

**Examples:** Number of heads in coin tosses, number of customers arriving, dice rolls.

### Definition 4.3 (Probability Mass Function)

For a discrete random variable $X$, the **probability mass function (PMF)** is:

$$p_X(x) = P(X = x)$$

**Properties:**
1. $p_X(x) \geq 0$ for all $x$
2. $\sum_{\text{all } x} p_X(x) = 1$
3. For any set $A$: $P(X \in A) = \sum_{x \in A} p_X(x)$

### Definition 4.4 (Continuous Random Variable)

A random variable $X$ is **continuous** if there exists a function $f_X(x)$ such that for any interval $[a,b]$:

$$P(a \leq X \leq b) = \int_a^b f_X(x) \, dx$$

**Key Property:** For continuous random variables, $P(X = x) = 0$ for any specific value $x$.

### Definition 4.5 (Probability Density Function)

For a continuous random variable $X$, the **probability density function (PDF)** $f_X(x)$ satisfies:

**Properties:**
1. $f_X(x) \geq 0$ for all $x$
2. $\int_{-\infty}^{\infty} f_X(x) \, dx = 1$
3. $P(a \leq X \leq b) = \int_a^b f_X(x) \, dx$

**Important:** $f_X(x)$ is NOT a probability; it's a density. Probabilities come from integration.

### Definition 4.6 (Cumulative Distribution Function)

The **cumulative distribution function (CDF)** of any random variable $X$ is:

$$F_X(x) = P(X \leq x)$$

**Properties:**
1. $0 \leq F_X(x) \leq 1$ for all $x$
2. $F_X$ is non-decreasing: if $x_1 < x_2$, then $F_X(x_1) \leq F_X(x_2)$
3. $\lim_{x \to -\infty} F_X(x) = 0$ and $\lim_{x \to \infty} F_X(x) = 1$
4. $F_X$ is right-continuous

**For discrete X:**
$$F_X(x) = \sum_{x_i \leq x} p_X(x_i)$$

**For continuous X:**
$$F_X(x) = \int_{-\infty}^x f_X(t) \, dt$$

$$f_X(x) = \frac{d}{dx} F_X(x) \text{ (where the derivative exists)}$$

## Worked Examples - Discrete Random Variables

### Example 1 (Simple): Coin Toss

**Problem:** Let $X$ be the number of heads in two fair coin tosses. Find the PMF and CDF of $X$.

**Solution:**

Sample space: $S = \{HH, HT, TH, TT\}$

Values of $X$: $\{0, 1, 2\}$

**PMF:**
- $p_X(0) = P(\{TT\}) = \frac{1}{4}$
- $p_X(1) = P(\{HT, TH\}) = \frac{2}{4} = \frac{1}{2}$
- $p_X(2) = P(\{HH\}) = \frac{1}{4}$

**CDF:**
$$F_X(x) = \begin{cases}
0 & x < 0 \\
\frac{1}{4} & 0 \leq x < 1 \\
\frac{3}{4} & 1 \leq x < 2 \\
1 & x \geq 2
\end{cases}$$

**Answer:** PMF: $p_X(0) = \frac{1}{4}$, $p_X(1) = \frac{1}{2}$, $p_X(2) = \frac{1}{4}$ $\square$

---

### Example 2 (Intermediate): Custom PMF

**Problem:** Let $X$ have PMF:
$$p_X(x) = \begin{cases}
\frac{x}{10} & x \in \{1, 2, 3, 4\} \\
0 & \text{otherwise}
\end{cases}$$

(a) Verify this is a valid PMF.
(b) Find $P(X > 2)$.
(c) Find the CDF.

**Solution:**

**(a)** Check properties:
1. $p_X(x) \geq 0$ for all $x$ ✓
2. $\sum p_X(x) = \frac{1}{10} + \frac{2}{10} + \frac{3}{10} + \frac{4}{10} = \frac{10}{10} = 1$ ✓

Valid PMF.

**(b)**
$$P(X > 2) = p_X(3) + p_X(4) = \frac{3}{10} + \frac{4}{10} = \frac{7}{10}$$

**(c)**
$$F_X(x) = \begin{cases}
0 & x < 1 \\
\frac{1}{10} & 1 \leq x < 2 \\
\frac{3}{10} & 2 \leq x < 3 \\
\frac{6}{10} & 3 \leq x < 4 \\
1 & x \geq 4
\end{cases}$$

**Answers:** (a) Valid, (b) $\frac{7}{10}$, (c) CDF given above $\square$

---

### Example 3 (Intermediate): Geometric Interpretation

**Problem:** A fair die is rolled. Let $X$ be the number shown. Find:
(a) The PMF
(b) $P(X \text{ is even})$
(c) $P(X \leq 4)$

**Solution:**

**(a)** For a fair die:
$$p_X(x) = \begin{cases}
\frac{1}{6} & x \in \{1, 2, 3, 4, 5, 6\} \\
0 & \text{otherwise}
\end{cases}$$

**(b)**
$$P(X \text{ even}) = p_X(2) + p_X(4) + p_X(6) = \frac{1}{6} + \frac{1}{6} + \frac{1}{6} = \frac{1}{2}$$

**(c)**
$$P(X \leq 4) = p_X(1) + p_X(2) + p_X(3) + p_X(4) = 4 \times \frac{1}{6} = \frac{2}{3}$$

**Answers:** (a) PMF is uniform on $\{1,2,3,4,5,6\}$, (b) $\frac{1}{2}$, (c) $\frac{2}{3}$ $\square$

## Worked Examples - Continuous Random Variables

### Example 4 (Simple): Uniform Distribution

**Problem:** Let $X$ be uniformly distributed on $[0, 2]$. Find:
(a) The PDF
(b) $P(0.5 \leq X \leq 1.5)$
(c) The CDF

**Solution:**

**(a)** For uniform distribution on $[a, b]$:
$$f_X(x) = \begin{cases}
\frac{1}{b-a} & a \leq x \leq b \\
0 & \text{otherwise}
\end{cases}$$

Here: $f_X(x) = \begin{cases}
\frac{1}{2} & 0 \leq x \leq 2 \\
0 & \text{otherwise}
\end{cases}$

**(b)**
$$P(0.5 \leq X \leq 1.5) = \int_{0.5}^{1.5} \frac{1}{2} \, dx = \frac{1}{2} \cdot 1 = \frac{1}{2}$$

**(c)**
$$F_X(x) = \begin{cases}
0 & x < 0 \\
\frac{x}{2} & 0 \leq x \leq 2 \\
1 & x > 2
\end{cases}$$

**Answers:** (a) $f_X(x) = \frac{1}{2}$ on $[0,2]$, (b) $\frac{1}{2}$, (c) CDF given above $\square$

---

### Example 5 (Intermediate): Exponential Distribution

**Problem:** Let $X$ have PDF $f_X(x) = 2e^{-2x}$ for $x \geq 0$ (and 0 otherwise). Find:
(a) Verify it's a valid PDF
(b) $P(X > 1)$
(c) The CDF

**Solution:**

**(a)** Check properties:
1. $f_X(x) \geq 0$ for all $x$ ✓
2. $\int_0^{\infty} 2e^{-2x} \, dx = 2 \cdot \left[-\frac{1}{2}e^{-2x}\right]_0^{\infty} = 2 \cdot \frac{1}{2} = 1$ ✓

Valid PDF.

**(b)**
$$P(X > 1) = \int_1^{\infty} 2e^{-2x} \, dx = \left[-e^{-2x}\right]_1^{\infty} = e^{-2} \approx 0.135$$

**(c)**
$$F_X(x) = \int_0^x 2e^{-2t} \, dt = \left[-e^{-2t}\right]_0^x = 1 - e^{-2x}$$

For $x \geq 0$. For $x < 0$, $F_X(x) = 0$.

$$F_X(x) = \begin{cases}
0 & x < 0 \\
1 - e^{-2x} & x \geq 0
\end{cases}$$

**Answers:** (a) Valid, (b) $e^{-2} \approx 0.135$, (c) $F_X(x) = 1 - e^{-2x}$ for $x \geq 0$ $\square$

---

### Example 6 (Challenging): Finding PDF from CDF

**Problem:** Let $X$ have CDF:
$$F_X(x) = \begin{cases}
0 & x < 0 \\
x^2 & 0 \leq x \leq 1 \\
1 & x > 1
\end{cases}$$

(a) Find the PDF.
(b) Calculate $P(0.2 \leq X \leq 0.8)$.

**Solution:**

**(a)** The PDF is the derivative of the CDF:
$$f_X(x) = \frac{d}{dx} F_X(x) = \begin{cases}
2x & 0 \leq x \leq 1 \\
0 & \text{otherwise}
\end{cases}$$

Verify: $\int_0^1 2x \, dx = \left[x^2\right]_0^1 = 1$ ✓

**(b)** Using the CDF:
$$P(0.2 \leq X \leq 0.8) = F_X(0.8) - F_X(0.2) = (0.8)^2 - (0.2)^2 = 0.64 - 0.04 = 0.6$$

Or using the PDF:
$$P(0.2 \leq X \leq 0.8) = \int_{0.2}^{0.8} 2x \, dx = \left[x^2\right]_{0.2}^{0.8} = 0.6$$

**Answers:** (a) $f_X(x) = 2x$ for $0 \leq x \leq 1$, (b) $0.6$ $\square$

## Exercises

### Exercise 1

Let $X$ have PMF: $p_X(1) = 0.2$, $p_X(2) = 0.3$, $p_X(3) = 0.5$. Find $E[X]$ and $P(X \geq 2)$.

**Hint:** $E[X] = \sum x \cdot p_X(x)$ and sum the relevant probabilities.

**Solution:**
$E[X] = 1(0.2) + 2(0.3) + 3(0.5) = 0.2 + 0.6 + 1.5 = 2.3$
$P(X \geq 2) = p_X(2) + p_X(3) = 0.3 + 0.5 = 0.8$ ✓

---

### Exercise 2

Let $X$ be uniformly distributed on $[1, 5]$. Find $P(2 \leq X \leq 3)$.

**Hint:** For uniform on $[a,b]$: $P(c \leq X \leq d) = \frac{d-c}{b-a}$.

**Solution:** $P(2 \leq X \leq 3) = \frac{3-2}{5-1} = \frac{1}{4}$ ✓

---

### Exercise 3

Let $X$ have PDF $f_X(x) = 3x^2$ for $0 \leq x \leq 1$. Find $P(X > 0.5)$.

**Hint:** Integrate the PDF from 0.5 to 1.

**Solution:** $P(X > 0.5) = \int_{0.5}^1 3x^2 \, dx = \left[x^3\right]_{0.5}^1 = 1 - 0.125 = 0.875$ ✓

---

### Exercise 4

For the random variable in Example 6, find $P(X < 0.5 \text{ or } X > 0.7)$.

**Hint:** Use complement or add the two probabilities (disjoint events).

**Solution:** $P(X < 0.5) = F_X(0.5) = 0.25$, $P(X > 0.7) = 1 - F_X(0.7) = 1 - 0.49 = 0.51$
Total: $0.25 + 0.51 = 0.76$ ✓

---

### Exercise 5

If $F_X(x) = 1 - e^{-\lambda x}$ for $x \geq 0$, find the PDF $f_X(x)$.

**Hint:** Take the derivative of the CDF.

**Solution:** $f_X(x) = \frac{d}{dx}(1 - e^{-\lambda x}) = \lambda e^{-\lambda x}$ for $x \geq 0$ ✓

## Applications

### Modeling Uncertainty
Random variables provide mathematical models for uncertain quantities in physics, engineering, and finance.

### Data Analysis
Discrete random variables model count data; continuous random variables model measurement data.

### Simulation
Random variables form the basis of Monte Carlo methods and stochastic simulation.

### Signal Processing
Noise in communication systems is modeled using continuous random variables.

## Common Mistakes

1. **Confusing PDF values with probabilities:** For continuous $X$, $f_X(x)$ is NOT $P(X=x)$ (which is always 0).

2. **Forgetting the normalization:** Always verify $\sum p_X(x) = 1$ or $\int f_X(x) \, dx = 1$.

3. **Improper handling of inequalities:** For discrete RVs, $P(X < x)$ vs $P(X \leq x)$ can differ; for continuous RVs, they're equal.

4. **Integration errors:** When finding probabilities from PDFs, carefully set integration limits.

## Summary

1. **Random variables** map outcomes to real numbers, enabling numerical analysis of random phenomena

2. **Discrete RVs** take countable values; characterized by PMF $p_X(x) = P(X=x)$

3. **Continuous RVs** take uncountable values; characterized by PDF $f_X(x)$ where probabilities come from integration

4. **CDF** $F_X(x) = P(X \leq x)$ applies to both discrete and continuous RVs

5. **Key relationships:**
   - Discrete: $F_X(x) = \sum_{x_i \leq x} p_X(x_i)$
   - Continuous: $F_X(x) = \int_{-\infty}^x f_X(t) \, dt$ and $f_X(x) = F_X'(x)$
