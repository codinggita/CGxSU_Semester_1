# Rank of a Matrix

## Learning Objectives

- Define the rank of a matrix rigorously
- Compute rank using row reduction to echelon form
- Understand relationships between rank, nullity, and dimensions
- Apply rank theorem to analyze linear systems
- Determine linear independence using rank
- Use rank in practical applications

## Prerequisites

- Row echelon form and RREF
- Elementary row operations
- Systems of linear equations
- Basic concepts of linear independence

## Definition and Computation

### Definition 4.1 (Rank of a Matrix)

The **rank** of a matrix $A$, denoted $\text{rank}(A)$ or $\rho(A)$, is the number of nonzero rows in any row echelon form of $A$.

**Equivalent definitions:**
1. The maximum number of linearly independent rows
2. The maximum number of linearly independent columns
3. The dimension of the row space (or column space)
4. The number of pivot positions in any echelon form

**Key insight:** All row echelon forms of $A$ have the same number of nonzero rows (though the actual rows may differ).

### Theorem 4.1 (Well-Definedness of Rank)

The rank of a matrix is well-defined: all row echelon forms of $A$ have the same number of nonzero rows.

**Proof sketch:** Pivot positions are uniquely determined by the structure of $A$, independent of the specific sequence of row operations. $\square$

### Algorithm 4.1 (Computing Rank)

**To find rank$(A)$:**
1. Reduce $A$ to row echelon form (REF) using elementary row operations
2. Count the number of nonzero rows
3. This count is rank$(A)$

**Note:** Can use either REF or RREF; both give the same rank.

## Properties of Rank

### Theorem 4.2 (Basic Properties)

For any matrix $A$ of size $m \times n$:

1. $0 \leq \text{rank}(A) \leq \min(m, n)$
2. $\text{rank}(A) = \text{rank}(A^T)$ (row rank equals column rank)
3. $\text{rank}(A) = 0 \iff A = O$ (zero matrix)
4. $\text{rank}(A) = n$ (maximum for $m \times n$) $\iff$ columns are linearly independent
5. $\text{rank}(A) = m$ (maximum for $m \times n$) $\iff$ rows are linearly independent

### Theorem 4.3 (Rank and Matrix Operations)

For matrices $A$ (size $m \times n$) and $B$ (compatible sizes):

1. $\text{rank}(cA) = \text{rank}(A)$ for any nonzero scalar $c$
2. $\text{rank}(AB) \leq \min(\text{rank}(A), \text{rank}(B))$
3. $\text{rank}(A + B) \leq \text{rank}(A) + \text{rank}(B)$
4. If $P$ and $Q$ are invertible, then $\text{rank}(PAQ) = \text{rank}(A)$

**Proof of (2):** Columns of $AB$ are linear combinations of columns of $A$, so column space of $AB$ is contained in column space of $A$. Thus rank$(AB) \leq$ rank$(A)$. Similarly, considering $(AB)^T = B^T A^T$, we get rank$(AB) \leq$ rank$(B)$. $\square$

### Definition 4.2 (Full Rank)

A matrix $A$ of size $m \times n$ has **full rank** if rank$(A) = \min(m, n)$.

- **Full row rank:** rank$(A) = m$ (all rows independent)
- **Full column rank:** rank$(A) = n$ (all columns independent)

### Theorem 4.4 (Rank and Invertibility)

A square matrix $A$ of size $n \times n$ is invertible if and only if rank$(A) = n$.

**Proof:** $A$ invertible $\iff \det(A) \neq 0 \iff$ RREF of $A$ is $I_n \iff$ rank$(A) = n$. $\square$

## Rank-Nullity Theorem

### Definition 4.3 (Nullity)

For an $m \times n$ matrix $A$, the **nullity** of $A$, denoted nullity$(A)$, is the dimension of the null space (solution space of $A\mathbf{x} = \mathbf{0}$).

**Equivalently:** nullity$(A)$ = number of free variables in the homogeneous system $A\mathbf{x} = \mathbf{0}$.

### Theorem 4.5 (Rank-Nullity Theorem)

For any $m \times n$ matrix $A$:

$$\text{rank}(A) + \text{nullity}(A) = n$$

where $n$ is the number of columns.

**Proof:** In RREF, rank$(A)$ = number of pivot columns, nullity$(A)$ = number of free variables = number of non-pivot columns. Since every column is either a pivot or non-pivot column, rank + nullity = total columns = $n$. $\square$

**Interpretation:** This fundamental theorem relates the dimension of the column space to the dimension of the null space.

## Rank and Linear Systems

### Theorem 4.6 (Rank and Solutions to $A\mathbf{x} = \mathbf{b}$)

For the system $A\mathbf{x} = \mathbf{b}$ where $A$ is $m \times n$:

1. **Consistent** $\iff$ rank$(A)$ = rank$([A | \mathbf{b}])$ (augmented matrix)
2. **Unique solution** $\iff$ rank$(A)$ = rank$([A | \mathbf{b}]) = n$
3. **Infinitely many solutions** $\iff$ rank$(A)$ = rank$([A | \mathbf{b}]) < n$

**Proof of (1):** System is inconsistent iff RREF has a row $[0\ 0\ \cdots\ 0 | c]$ with $c \neq 0$ iff rank$([A | \mathbf{b}])$ = rank$(A) + 1$. $\square$

### Corollary 4.1 (Existence for All $\mathbf{b}$)

The system $A\mathbf{x} = \mathbf{b}$ has a solution for **every** $\mathbf{b} \in \mathbb{R}^m$ if and only if rank$(A) = m$ (full row rank).

## Worked Examples

### Example 1 (Simple): Computing Rank

**Problem:** Find rank$(A)$ for $A = \begin{bmatrix} 1 & 2 & 3 \\ 2 & 4 & 6 \\ 1 & 1 & 2 \end{bmatrix}$.

**Solution:**

Reduce to REF:

$R_2 - 2R_1 \to R_2$, $R_3 - R_1 \to R_3$:
$$\begin{bmatrix} 1 & 2 & 3 \\ 0 & 0 & 0 \\ 0 & -1 & -1 \end{bmatrix}$$

$R_2 \leftrightarrow R_3$:
$$\begin{bmatrix} 1 & 2 & 3 \\ 0 & -1 & -1 \\ 0 & 0 & 0 \end{bmatrix}$$

Number of nonzero rows = 2.

**Answer:** rank$(A) = 2$ $\square$

---

### Example 2 (Simple): Rank and Nullity

**Problem:** For $A = \begin{bmatrix} 1 & 2 & 3 & 4 \\ 0 & 1 & 2 & 3 \end{bmatrix}$ (already in REF), find rank$(A)$ and nullity$(A)$.

**Solution:**

**Rank:** Two nonzero rows, so rank$(A) = 2$.

**Nullity:** By rank-nullity theorem:
$$\text{nullity}(A) = n - \text{rank}(A) = 4 - 2 = 2$$

**Answer:** rank$(A) = 2$, nullity$(A) = 2$ $\square$

---

### Example 3 (Intermediate): Full Rank Determination

**Problem:** Determine if $A = \begin{bmatrix} 1 & 2 \\ 3 & 4 \\ 5 & 6 \end{bmatrix}$ has full column rank.

**Solution:**

For full column rank, need rank$(A) = 2$ (number of columns).

Reduce to REF:

$R_2 - 3R_1 \to R_2$, $R_3 - 5R_1 \to R_3$:
$$\begin{bmatrix} 1 & 2 \\ 0 & -2 \\ 0 & -4 \end{bmatrix}$$

$R_3 - 2R_2 \to R_3$:
$$\begin{bmatrix} 1 & 2 \\ 0 & -2 \\ 0 & 0 \end{bmatrix}$$

rank$(A) = 2$ = number of columns.

**Answer:** Yes, $A$ has full column rank. Columns are linearly independent. $\square$

---

### Example 4 (Intermediate): Rank of Product

**Problem:** If rank$(A) = 3$ and rank$(B) = 2$ where $A$ is $4 \times 5$ and $B$ is $5 \times 6$, what can you say about rank$(AB)$?

**Solution:**

By Theorem 4.3(2):
$$\text{rank}(AB) \leq \min(\text{rank}(A), \text{rank}(B)) = \min(3, 2) = 2$$

**Answer:** rank$(AB) \leq 2$. We cannot determine the exact value without more information, but it cannot exceed 2. $\square$

---

### Example 5 (Challenging): Rank and Consistency

**Problem:** For what values of $k$ does the system have a solution?
$$\begin{cases}
x + y + z = 1 \\
2x + 3y + 2z = 3 \\
x + y + (k^2 - 3)z = k
\end{cases}$$

**Solution:**

Augmented matrix:
$$\left[\begin{array}{ccc|c} 1 & 1 & 1 & 1 \\ 2 & 3 & 2 & 3 \\ 1 & 1 & k^2-3 & k \end{array}\right]$$

$R_2 - 2R_1 \to R_2$, $R_3 - R_1 \to R_3$:
$$\left[\begin{array}{ccc|c} 1 & 1 & 1 & 1 \\ 0 & 1 & 0 & 1 \\ 0 & 0 & k^2-4 & k-1 \end{array}\right]$$

**Case 1:** $k^2 - 4 \neq 0$ (i.e., $k \neq \pm 2$)

Then rank$(A) = 3 =$ rank$([A|\mathbf{b}])$. System is consistent.

**Case 2:** $k = 2$

Row 3 becomes $[0\ 0\ 0 | 1]$. Inconsistent (rank$(A) = 2$, rank$([A|\mathbf{b}]) = 3$).

**Case 3:** $k = -2$

Row 3 becomes $[0\ 0\ 0 | -3]$. Inconsistent.

**Answer:** System has a solution if and only if $k \neq 2$ and $k \neq -2$. $\square$

---

### Example 6 (Challenging): Rank-Nullity Application

**Problem:** Let $A$ be a $5 \times 7$ matrix with rank$(A) = 3$. How many free variables does the homogeneous system $A\mathbf{x} = \mathbf{0}$ have? What is the dimension of the solution space?

**Solution:**

By rank-nullity theorem:
$$\text{nullity}(A) = n - \text{rank}(A) = 7 - 3 = 4$$

Number of free variables = nullity$(A) = 4$.

Dimension of solution space (null space) = nullity$(A) = 4$.

**Answer:** 4 free variables; solution space is 4-dimensional. $\square$

## Applications

### Network Theory

Rank of incidence matrix determines number of independent loops in a network (circuits, graphs).

### Control Theory

Controllability and observability of linear systems depend on ranks of specific matrices.

### Computer Graphics

Rank determines dimensionality of affine transformations; singular transformations (rank $< n$) project to lower dimensions.

### Data Science

In PCA (Principal Component Analysis), effective rank (number of large singular values) indicates intrinsic dimensionality of data.

### Coding Theory

Rank of generator matrix determines dimension of linear code; relates to error-correction capability.

## Common Mistakes

1. **Confusing row and column count with rank:** Rank is number of *nonzero* rows in REF, not total rows.

2. **Assuming rank$(A + B) =$ rank$(A) +$ rank$(B)$:** This is an inequality, not equality in general.

3. **Forgetting rank-nullity:** Always verify that rank + nullity = number of columns.

4. **Inconsistent vs. rank deficient:** Inconsistent means no solution; rank deficient means infinitely many solutions (if consistent).

5. **Confusing $[A|\mathbf{b}]$ and $A$:** For consistency, must check rank of *augmented* matrix.

## Summary

1. **Rank:** Number of nonzero rows in REF; equals number of pivot positions

2. **Properties:**
   - $0 \leq$ rank$(A) \leq \min(m, n)$
   - rank$(A) =$ rank$(A^T)$
   - rank$(AB) \leq \min($rank$(A)$, rank$(B))$

3. **Rank-Nullity Theorem:** rank$(A) +$ nullity$(A) = n$ (number of columns)

4. **Full rank:**
   - Square $n \times n$: rank$(A) = n \iff A$ invertible
   - General $m \times n$: rank$(A) = \min(m,n)$

5. **Linear systems:**
   - Consistent $\iff$ rank$(A) =$ rank$([A|\mathbf{b}])$
   - Unique solution $\iff$ consistent and rank$(A) = n$
